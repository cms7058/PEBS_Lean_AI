# 精益生产基础概念

精益生产（Lean Production）源自丰田生产方式（TPS, Toyota Production System），核心目标是**用更少的资源创造更多客户价值**。Womack 与 Jones 在《Lean Thinking》中提炼出五大原则：

## 五大原则

1. **定义价值（Value）**：从客户视角识别什么值得付钱，什么是无谓消耗。客户不会为返工、库存、等待付费。
2. **识别价值流（Value Stream）**：把产品从原材料到客户手中的所有活动列出来，区分增值（VA）、必要非增值（NVA-N）、纯浪费（NVA-W）。常用工具是价值流图（VSM）。
3. **流动（Flow）**：让价值连续流动，消除工序间停顿、批次堆积、等待。一个流（One-Piece Flow）是理想状态。
4. **拉动（Pull）**：由下游（最终客户）需求触发上游生产，而非按预测推动。看板（Kanban）是经典拉动信号机制。
5. **尽善尽美（Perfection）**：通过持续改善（Kaizen）追求零浪费、零缺陷、零库存的理想状态——是方向而非终点。

## 三大基石

- **稳定（Stability）**：标准作业、设备可用、质量稳定、人员胜任。无稳定无精益。
- **准时化（JIT, Just-In-Time）**：在需要的时间、按需要的数量、提供需要的物品。
- **自働化（Jidoka）**：异常自动停机/暴露问题，让缺陷无法流向下工序。安灯系统（Andon）是典型实现。

## 常见误区

- 精益 = 裁员：错。精益旨在解放重复劳动让人去做改善，与裁员无关。
- 精益 = 单纯的工具集（5S/SMED/看板）：错。工具是手段，**思维方式与文化**才是核心。
- 精益只适合大批量制造：错。医院、软件开发（Lean Software / DevOps）、服务业都已成功应用。

## 与六西格玛的关系

- 精益侧重**消除浪费、提高速度**（Time）。
- 六西格玛侧重**减少变异、提升质量**（Quality）。
- 实践中常融合为 **Lean Six Sigma**，用 DMAIC 框架 + 精益工具协同改善。
