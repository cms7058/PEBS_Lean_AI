# 5S 现场管理

5S 是精益现场管理的起点，源自日本的整理（Seiri）、整顿（Seiton）、清扫（Seiso）、清洁（Seiketsu）、素养（Shitsuke）。看似基础，是所有进一步改善的土壤。

## 5 个 S 的内容

### 1S 整理（Seiri）—— 区分要 / 不要
- 区分必要 / 不必要的物品，移除不要的
- 红牌作战（Red Tag）：可疑物品挂红牌等待处置
- 输出：现场只留下"必要数量的必要物品"

### 2S 整顿（Seiton）—— 定置定位
- 给每件物品一个固定位置：定品、定位、定量
- 三定法 + 影子板（Shadow Board）+ 颜色管理
- 标准：30 秒内任何人能找到任何东西

### 3S 清扫（Seiso）—— 清扫即点检
- 不仅打扫，更是发现异常（漏油、松动、磨损）
- 清扫责任区划分到人
- 与设备自主保全结合

### 4S 清洁（Seiketsu）—— 维持标准化
- 把前 3S 的成果固化为标准（清洁基准书、点检表）
- 可视化管理（看板、警示线、定置图）

### 5S 素养（Shitsuke）—— 习惯化
- 全员遵守标准成为习惯
- 5S 巡查、改善提案、班组荣誉激励
- 文化层面：对秩序与规则的敬畏

## 为什么 5S 是基础

| 5S 缺失会导致 | 影响 |
|---|---|
| 找工具/物料浪费时间 | 节拍不稳定 |
| 异常被掩盖 | 故障突发、缺陷漏出 |
| 安全隐患 | 滑倒、磕碰、起火 |
| 改善无据可依 | "改"完没基线对比 |
| 看上去乱糟糟 | 客户审厂减分 |

## 推进步骤

1. **承诺与培训**：领导层公开承诺、全员 5S 培训
2. **试点车间**：选 1–2 个工段先做样板
3. **5S 大扫除日**：集中投入 1–3 天清空、分类、整顿
4. **标准化**：制定清洁基准书、定置图、点检表
5. **巡检评分**：周巡 + 月评比，红黄绿三色看板
6. **持续改善**：每月一个改善主题（如"工具看板周")

## 5S 评估表（简化版）

每项 0–5 分，5 个 S 共 25 分，可按工段周评比。

- 1S 整理：是否仍能找到不必要物品？
- 2S 整顿：物品是否在定置位置？标识是否清晰？
- 3S 清扫：地面、设备、工装是否干净？
- 4S 清洁：是否有清洁标准书并被遵守？
- 5S 素养：员工是否自觉维持？

## 与其他精益工具的关系

- **TPM 自主保全** ⊃ 3S 清扫
- **VSM** 现场观察的前提是 5S 已建立
- **目视化管理（Visual Management）** 是 4S 的延伸
- **SMED** 工装就近化、影子板都依赖 2S 整顿

## 常见误区

- "5S = 大扫除"——忽略整顿与标准化
- 只在评比前突击做——伪 5S
- 墙上挂满标语，现场依然乱——形式主义
- 把不要的东西藏起来而非移除——红牌作战不彻底
