# SMED 快速换型

SMED（Single-Minute Exchange of Die）由新乡重夫提出，目标是把换型/换模时间压缩到 **10 分钟以内**（"单分钟" = 个位数分钟）。是缩小批量、实现精益流动的关键基础。

## 为什么换型重要

- 换型长 → 必须用大批量摊薄换型成本 → 大批量 → 库存高、Lead Time 长、缺陷不易暴露
- 换型短 → 小批量经济 → 一个流可行 → 库存↓、响应↑、质量↑

经典数据：丰田曾把冲压机换型从 4 小时压缩到 3 分钟，使小批量多品种生产成为可能。

## 内部 / 外部作业的区分

SMED 的核心洞察：**把"内部作业"转换为"外部作业"**。

| 类型 | 定义 | 例子 |
|---|---|---|
| 内部作业 | 必须在设备**停机**时完成 | 拆装模具、对刀、试切 |
| 外部作业 | 可在设备**运行**时完成 | 准备工具、预热模具、搬运 |

## SMED 四步法

### 第 1 步：观察并区分内 / 外作业
现场拍视频，逐项标注每个动作属内部还是外部。**通常 30–50% 的"内部作业"其实可以外部化**。

### 第 2 步：把内部作业外部化
- 模具预热提前到上一品种生产时进行
- 工具、量具提前到位（影子板 / Set-up cart）
- 物料按生产顺序预备齐套

### 第 3 步：精简内部作业
- 标准化紧固件（同规格螺栓、同高度模具）
- **取消螺栓**：用快速夹具、磁吸、销定位
- 平行作业：两人协同，单人换型时间 = 双人时间 / 2
- 消除调整：用定位销、刻度尺、首件免试切

### 第 4 步：精简外部作业
- 工装车标准化、就近化
- 物料 Kit 化（按订单整套备料）
- 信息化：MES 提前下发换型指令

## 量化效果

| 阶段 | 典型时间压缩 |
|---|---|
| 仅做内/外区分 | -30% |
| 完成内部外部化 | -50% |
| 三步法走完 | -75% |
| 极致优化 | -90% 以上 |

## 与其他工具的关系

- **VSM**：找出哪些工序换型最痛
- **5S**：工装标准化、可视化是 SMED 的前提
- **TPM**：设备稳定性差会让换型时间不稳定
- **Heijunka**：换型短才能小批量均衡生产
- **OEE**：换型时间是性能损失的重要来源

## 实施常见障碍

- 操作工抗拒：习惯了大批量节奏
- 设备老旧：定位精度差，无法快速对位
- 计划部门不配合：仍按"大批量摊薄"思路排产
- 缺乏现场视频回放习惯：靠经验拍脑袋改不下去

## 改善案例参考

- 注塑机：8h → 12min（更换快速换模系统 + 模具预热 + 平行作业）
- 冲压线：120min → 10min（消除螺栓 + 标准模高 + Set-up cart）
- 包装线：45min → 5min（标准化模具规格 + 刻度定位）
